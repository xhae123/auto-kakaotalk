사회성 떨어지는 개발자들을 위한 인간관계 자동화 도구에요. 카톡창 열기 싫고, 답장하자니 뭐라 써야 할지 모르겠고, 안 할 수도 없고. 이걸 AI 한테 맡겨봤어요. 제가 저를 복제하여 카톡 앞에 앉히는 겁니다.

## Features

카톡을 대신 해줘요. 그게 전부예요.

- 등록할 때 AI 가 과거 카톡을 읽고 **저랑 상대 관계를 한 번 분석해서 보고**해줘요. "얘는 너랑 10년 친구고, 주도권은 살짝 얘가 더 쥐고 있고, 꿈 얘기 시작하면 네가 보통 짧게 끊더라" 이런 식으로요. 그거 보고 저는 교정해요. "이 부분은 맞고, 저 부분은 틀렸고, 얘한텐 이런 건 아예 답하지 마."
- 조율 끝나면 그 뒤로는 묻지 않아요. 맥북 켜져 있는 동안 메시지가 오면 AI 가 **알아서 나인척 하고 보내요.** 
- 전송한 건 한 줄로만 알려줘요. "철수한테 '저녁 뭐 먹었어' 보냄."

## Motivation

사회성을 기르는 거보다 AI에게 인간관계 외주 주는 게 공학적으로 더 쉬웠어요.

답장 한 줄 쓰는 데 30분 고민하는 걸 매번 반복할 바엔, 한 번에 풀어버리고 싶었거든요. "내가 쓸 법한 답장" 에는 패턴이 있어요. 패턴은 학습되고, 학습되면 자동이 돼요. 인간관계를 포기한 게 아니에요. 유지 노동만 넘긴 거예요.

비슷한 고민 있으면 한 번 써보세요!

## Demo

설치하고 Claude Code 에서 실행하면, AI가 다음과 같이 알아서 나를 복제하고, 카톡을 대신 해줘요.

```
[AI]  지수랑 과거 대화 1,847개 읽었어. 베이스 준비됐고,
      앞으로 오가는 대화로 계속 배울게. 시작한다.
```

그리고 자리 비운 사이 이런 게 일어나요.

```
────────────  당신이 카톡을 한동안 안 본 사이  ────────────

10:23   엄마     밥 먹었니?
10:24   당신     응 방금 먹었어     ← AI

13:47   민수     오늘 뭐해
13:49   당신     집임 ㅋㅋ 왜       ← AI
13:50   민수     카페 갈래
13:52   당신     ㄱㄱ 강남역에서     ← AI

20:15   직장상사 내일 보고서 확인 부탁합니다
20:16   당신     네 내일 아침에 확인하고 드리겠습니다   ← AI

────────────────────────────────────────────────────
```

엄마한테는 엄마 말투, 직장상사한테는 직장상사 말투. 맥북만 켜져 있으면 돼요. 이 대화가 오간 걸 당신이 아는 건, 나중에 카톡을 직접 열었을 때예요.

## How It Works

데몬 없고 서버 없어요. Claude Code 세션 자체가 런타임이에요. 창 닫으면 그대로 멈춰요. 끄는 방법이 창 닫기인 게 이 도구의 안전장치예요.

3분마다 카톡을 들여다보고, 새 메시지가 있으면 Claude 가 답장 여부와 톤, 내용을 한 번의 판단 안에서 정해요. 감정 분류기 따로, 생성기 따로 두지 않았어요. "됐어" 세 글자가 화났다는 건지 괜찮다는 건지는 중간 레이어에 숫자를 붙이는 순간 사라지거든요.

승인 루프는 일부러 뺐어요. 매 메시지마다 "보낼까?" 묻는 순간 사용자가 그 물음을 귀찮아서 결국 도구 자체를 안 쓰게 돼요. 그러면 원점이잖아요. 대신 등록 시점에 딱 한 번, 과거 대화를 분석해서 "이 관계는 이렇고, 말투는 이렇고, 이런 건 피하고" 를 사용자와 합의해요. 그 합의를 페르소나 파일에 박아두고, 루프는 그걸 따라요. 관계가 변해서 뉘앙스를 다시 잡고 싶으면 "맹구 다시 분석해줘" 한 마디면 재조율.

전송은 `queued → sending → sent` 2단계로 기록해요. 한쪽으로 비대칭하게 편향시켰어요. "못 보냈다" 보다 "두 번 보냈다" 가 훨씬 나쁘거든요. 세션이 중간에 죽으면 stuck 된 건은 무조건 failed 처리. 진짜로 보냈으면 다음 poll 에서 outbound 로 되돌아와요. 같은 메시지가 두 번 나가는 일은 없어요.

설계 결정과 실패 사례는 블로그에.

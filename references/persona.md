# 페르소나

## 한 줄 정의

`state/personas/<chat_id>.md` 는 **사용자가 명시적으로 주는 지시**를 담는 자유형식 파일이다. 관찰된 말투가 아니라, 관찰된 것과 **다르게** 하고 싶을 때 쓰는 오버라이드.

## 왜 있는가

과거 대화만으로는 사용자의 의도를 전부 알 수 없다. 예를 들어:

- "이 사람한테는 좀 더 친절하게 해줘"
- "이 단톡방에서는 반말이 섞이던데, 나는 앞으로 존댓말로만 가고 싶어"
- "XX 라는 표현 절대 쓰지 마"
- "회사 동료한테는 이모지 쓰지 말자"

과거 대화에 안 드러나거나, 과거와 다르게 가고 싶은 경우. 이런 걸 담는다.

비어 있어도 된다. 비어 있으면 관찰된 말투만으로 답한다.

## 어떻게 채워지나

- **등록 직후**: 빈 파일로 생성됨.
- **사용자가 말로 부탁**: "그 방에선 존댓말로 해줘" → Claude 가 세션 안에서 해당 파일에 한 줄 추가.
- **사용자가 직접 편집**: 그냥 마크다운이라 편집기 열어서 쓰면 됨.

## 형식

자유형식. 아래는 예시.

```md
# 엄마 용 지시사항

- 길이는 짧게. 평소보다도 더 짧게.
- "사랑해" 같은 표현 쓰지 말 것 (어색함).
- 용돈 얘기 나오면 직접 답하지 말고 skip.
```

## Claude 는 매 cycle 에 이걸 읽는다

답장을 드래프트할 때마다 Claude 가 `Read` 로 persona.md 를 열고, 최근 30개 대화와 합쳐 판단. 파일이 비어 있으면 그 단계는 스킵.

## 불변 조건

없다. 자유형식 텍스트 파일이고, 사용자 소유다. Claude 는 사용자가 명시적으로 요청할 때만 수정한다.
